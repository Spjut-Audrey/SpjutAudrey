Using JavaScript in Events

How to trigger:

  • Mouse Events
    • onlick
    • ondblclick
    • onmouseover
  • Keyboard Events
    • onkeypress
  • Frame/Object Events
    • onload
    • onresize
    • onscroll
  • Form Events
    • onfocus
    • oninput
    • onsubmit
  • Drag Events
    • ondrag
    • ondrop
  • Clipboard Events
    • oncopy
    • onpaste
  • Print Events
    • onafterprint
    • onbeforeprint
  • Media Events
    • onplay
    • onpause
    • onwaiting
  • Animation Events
    • animationend
    • animationiteration
    • animationstart
  • Transition Events
    • transitionend
  • Touch Events
    • ontouchstart // end
    • ontouchmove

Touch Events Browser Support

General Support for Touch events:

  • FireFox
  • Chrome
  • Opera
  • iOS Safari
  • Android Browser
  • Chrome (Android)

Not Supported:

  • IE
  • Edge
    • Can be enabled using "about:flags"
  • Safari
  • Opera Mini

Important to know:

  • Android 2.3 and below - no multi-touch support
  • Most browsers don't completely support all touch events. Chrome v65 and Opera v50 have 100% support.
