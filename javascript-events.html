How to trigger:

  • Mouse Events

    • ondblclick
    • onmouseover
  • Keyboard Events
    • onkeypress
  • Frame/Object Events
    • onload
    • onresize
    • onscroll
  • Form Events
    • onfocus
    • oninput
    • onsubmit
  • Drag Events
    • ondrag
    • ondrop
  • Clipboard Events
    • oncopy
    • onpaste
  • Print Events
    • onafterprint
    • onbeforeprint
  • Media Events
    • onplay
    • onpause
    • onwaiting
  • Animation Events
    • animationend
    • animationiteration

  • Transition Events
    • transitionend
  • Touch Events
    • ontouchstart // end
    • ontouchmove
