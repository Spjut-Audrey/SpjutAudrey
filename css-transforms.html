CSS Transforms

It's fun to play with and easy to create using CSS. This page shows the different things you can do with CSS Transforms.

Translating can use different planes. TranslateX-Z are a few ways to use this. (transform: translate(30px,10px); used on this)
You can scale the boxes you transform using the same types of planes similar to translating. (transform: scaleY(.65); used on this div.)
Rotating is a little different. Use angles(deg) to to specify how much you want it to rotate. X-Z is also applicable. (transform: rotateX(180deg);) used.
Skew is similar to rotating in that it also uses angles and planes. (transform: skewX(30deg);) used.
